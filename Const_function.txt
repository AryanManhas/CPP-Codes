Const Function

We can make functions const by using const keyword in the function decleration
We create functions an const so that accidental changes to objects are avoided

Constant member functions are those functions that are fenied permissions to change the values of the data member of their class.
const functions can be called by the non const objects.

When a function is decleared as const, it can be called on any type of object.
Non const functions can only be called by non-const objects.
